# Оценка рыночной цены для шин и дисков

Проект создаёт веб-сайт для расчёта конечной рыночной стоимости автомобильных шин и дисков.

## Архитектура

- Python/FastAPI — бэкенд
- PostgreSQL — база данных
- Jinja2 + Bootstrap — простой фронтенд
- SQLAlchemy — ORM
- Модуль ML для расчёта рекомендованной цены
- Парсеры Ozon и Wildberries — сбор данных с поисковых страниц и JSON-встраиваний сайта

## Установка

1. Убедитесь, что [`uv`](https://github.com/astral-sh/uv) установлен:

```bash
uv --version
```

2. Создайте виртуальное окружение и установите зависимости:

```bash
uv venv
uv sync
```

3. Настройте базу данных — создайте файл `.env` с параметром:

```env
DATABASE_URL=postgresql+psycopg2://user:password@localhost:5432/tire_price_db
SECRET_KEY=change_this_secret
DEBUG=True
```

## Запуск

```bash
uv run uvicorn app.main:app --reload
```

Откройте `http://127.0.0.1:8000/` в браузере.
